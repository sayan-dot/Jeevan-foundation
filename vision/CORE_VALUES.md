# Purpose

This document defines the core values of Jeevan.

Core values represent the principles that guide every decision, regardless of changes in technology, products, or business strategy.

Every employee, designer, developer, and stakeholder should understand and uphold these values.

---

# Our Core Values

## 1. Human First

Technology exists to serve people.

Every product decision should begin by asking how it improves the user's life, rather than how advanced or innovative it appears.

People should never have to adapt to technology.

Technology should adapt to people.

---

## 2. Accessibility by Default

Accessibility is not an additional feature.

It is a requirement.

Every product should be designed so that people of different ages, abilities, languages, and levels of digital literacy can use it with confidence.

Good accessibility is good design.

---

## 3. Simplicity Over Complexity

The simplest solution is often the best solution.

We value clarity over cleverness.

Every unnecessary step, screen, or interaction should be questioned.

If something requires lengthy explanations, it should probably be redesigned.

---

## 4. Trust Above Everything

Healthcare depends on trust.

Users trust us with some of the most important aspects of their lives.

That trust must never be compromised.

Every decision should strengthen trust through honesty, transparency, reliability, and respect for privacy.

---

## 5. Empathy in Every Decision

Behind every notification is a real person.

Behind every reminder is someone's health.

Behind every feature is someone's family.

We design with empathy before efficiency.

---

## 6. Build With Purpose

Every feature must solve a real problem.

We do not build features because they are popular.

We build features because they improve lives.

Purpose always comes before innovation.

---

## 7. Learn Continuously

Healthcare evolves.

Technology evolves.

People evolve.

Jeevan must continue learning through research, user feedback, and continuous improvement while remaining true to its mission.

---

# Decision Framework

When making difficult decisions, ask:

- Does this help people?
- Does this simplify healthcare?
- Does this improve accessibility?
- Does this strengthen trust?
- Does this align with our mission?

If the answer is no, the decision should be reconsidered.

---

# Non-Negotiables

These principles should never be compromised.

- User trust
- Accessibility
- Privacy
- Simplicity
- Human-centered design

Regardless of business pressure, deadlines, or trends.

---

# Closing Statement

Technology will continue to change.

Healthcare will continue to evolve.

These values should remain constant.

They define not only the products we build, but also the company we aspire to become.
