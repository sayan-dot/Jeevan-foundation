# Purpose

Typography is one of the primary ways Jeevan communicates with its users.

The goal of typography is not to create visual interest.

Its purpose is to make information easy to read, understand, and act upon.

Healthcare information should always be clear before it is beautiful.

---

# Philosophy

Reading should never require effort.

Users should never struggle because text is too small, too dense, or poorly organized.

Typography should quietly support communication rather than draw attention to itself.

If users notice the typography before they notice the information, the design should be reconsidered.

---

# Principles

- Readability over decoration.
- Consistency over creativity.
- Hierarchy before styling.
- Large enough for confidence.
- Minimize unnecessary reading.

---

# Typography Roles

Typography is organized by purpose rather than size.

| Role | Purpose |
|------|---------|
| Display | Hero sections and onboarding |
| Heading | Screen titles |
| Subheading | Section titles |
| Body | General information |
| Label | Buttons and form labels |
| Caption | Supporting information |
| Helper | Additional context |

---

# Hierarchy

Every screen should clearly communicate:

1. What this screen is about.
2. What information is important.
3. What action should be taken next.

Typography should establish this hierarchy without relying on excessive styling.

---

# Readability Guidelines

Typography should remain readable in all supported languages.

Guidelines:

- Use generous line spacing.
- Avoid long paragraphs.
- Prefer sentence case.
- Maintain sufficient contrast.
- Avoid decorative fonts.

---

# Accessibility

Typography should support users with:

- Reduced vision
- Reading difficulties
- Cognitive overload
- Limited digital literacy

The interface should remain readable without requiring users to zoom or strain.
